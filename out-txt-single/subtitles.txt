สวัสดีครับในคลิปวีดีโอนี้นะครับเราจะมาพูดถึง html Table กัน html Table นะครับคือการเขียน Table หรือตารางด้วย html ต้องบอกก่อนนะครับว่าตารางใน hml เนี่ยเป็นอะไรที่ใช้เยอะกันมากๆนะครับด้วยความที่ Structure ของตารางน้อยที่มีคอลัมน์มี low มันทำให้การจัดเก็บข้อมูลเนี่ยเคลียร์ขึ้นนะครับอ่านง่ายๆขึ้นเนาะแล้วก็แสดงผลให้ User เนี่ยอ่านแล้วเข้าใจได้ง่ายครับอีกอย่างนะครับเจ้าตัวของ Table ด้วยความที่มันทำให้ Structure ของข้อมูลที่โชว์บนหน้าเว็บเนี่ยอ่านง่ายเข้าถึงง่ายทำให้คนส่วนใหญ่นะครับดีไซน์หน้าเว็บเนี่ยให้อ้างอิงกับตาราง Table ไม่ว่าจะเป็นการโชว์ข้อมูลในอะไรต่างๆที่จำเป็นต้องอยู่ใน Structure ของ Table ก็จะนิยมใช้ Table layout นะครับในการสร้างหน้านั้นขึ้นมาผมยกตัวอย่างนะครับอย่างหน้านี้เนาะเราจะเห็นนะครับว่าเขาเนี่ยก็มีการสร้างตารางขึ้นมาเพื่อที่จะดูข้อมูลได้ง่ายไม่ว่าจะเป็นจำนวนของ City ใช่ไหมครับก็สร้างเป็นหัวคอลัมน์เป็น Country หรือว่ามีการใช้หัวคอลัมน์ที่เป็นการเมิดคนหลายๆคอลัมน์รวมกันนะครับเหมือนหลายๆคอลัมน์รวมกันแล้วก็มีชายที่เป็นแต่ละคอลัมน์คอลัมน์ซึ่งมันทำให้การอ่านของข้อมูลเนี่ยมีความอ่านแล้วเข้าใจมากขึ้นเนาะเพราะว่าปกติเราอ่านข้อมูลใน Table เนี่ยเราก็จะหาจุดตัดของแกน x แกน Y ใช่ไหมครับอย่างเช่นถ้าผมต้องการดูว่า City ไหนนะครับที่มี area เท่าไหร่เนี่ยเราก็แค่หาจากโรงแล้วก็ดูคอลัมน์หรือข้อมูลที่เราต้องการนะครับอันนี้คือจุดเด่นของตารางที่เขาเนี่ยนิยมมาสร้างในเว็บอันนี้จะเป็นตารางที่เราเห็นชัดๆเลยเนาะว่ามันคือตารางแต่ก็มีบาง Structure นะครับหรือบาง layout ของหน้าเว็บไซต์ที่มองเผินๆเนี้ยมันไม่ใช่ตารางนะแต่มันถูกเขียนด้วย html Table นะครับอ่าผมพาไปดูเว็บนี้ทุกคนเห็นอะไรไหมครับหน้านี้ไม่เหมือนเป็นตารางเลยถูกไหมครับไม่เหมือนเป็นตารางเลยเหมือนแบบว่าเอาแต่ละอีเมลมาวางต่อกันเฉยๆแต่ทุกคนรู้ไหมครับว่านะส่วนนี้เขาสร้างขึ้นมาด้วย htm อ่ะเราลองไปเช็คดูก่อนผมจะจิ้มไปที่ตรงนี้นะครับอ่าจิ้มไปทุกคนเห็นอะไรไหมครับ Table เห็นมั้ยเห็นมั้ยครับนี่แหละคือตัวบ่งบอกว่าอันเนี้ยเขียนด้วยตารางนะครับแล้วใน Table เนี่ยก็จะมี Element รายละเอียดต่างๆที่เอาไว้สร้าง Table หรือสิ่งที่อยู่ใน Table อีกทีที่เราจะได้เรียนกันนะครับที่เราจะได้เรียนกันเราลองไล่ดูซิว่าเห็นมั้ยเค้าเค้าเขียนด้วย Table เนี่ยเขาก็จะแบ่งนะครับคอลัมน์ออกเป็นกี่คอลัมน์เราก็ลองเขาก็แบ่งคอลัมน์ออกนะครับเห็นมั้ยคอลัมน์แรกคอลัมน์ที่ 2 เพราะร้านที่ 3 นะครับอ่าเรามี 3 เนี่ยเขาก็จะทำเป็นแนวนอนถูกไหมครับทำเป็นแนวนอนเนี่ยเห็นมั้ยอันนี้แหละครับคือการใช้ html Table นะครับมาประยุกต์ใช้กับ layout หลายๆอย่างที่เราต้องการแสดงบนเว็บเรานะครับเพราะว่า Structure ของ Table เนี่ยมันค่อนข้างแคลอรี่คืออ่านง่ายดูง่ายและอีกเหตุผลหนึ่งที่เราจะต้องคอนเซอกันนิดนึงเนาะคือ html Table นะครับมันดูง่ายใช่ไหมแต่ว่าการที่จะเขียน Table ขึ้นมาด้วย hml เนี่ยมันมี Element หลายอย่างมากเลยที่เราจะต้องมาใช้ร่วมกันนะครับเราจะต้องใช้ประมาณ 5-10 different Element คือ Element ต่างกันที่เราจะสร้าง Table ขึ้นมาสัก Table เนี่ยอันนี้ก็เป็นโจทย์ของเราแล้วว่าเราอาจจะต้องโฟกัสนิดนึงนะครับว่าการสร้างตารางเนี่ยมันประกอบไปด้วยอีเมลอะไรบ้างแต่ละ Element เนี่ยทำหน้าที่อะไรนะครับถ้าพร้อมแล้วก็เดี๋ยวเราไปลุยกันเลยครับ
